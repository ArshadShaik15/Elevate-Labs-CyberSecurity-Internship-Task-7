### Summary:<br>

<p align="left">The goal was to <strong>Identify and Remove Suspicious Browser Extensions</strong> by performing an audit of my installed browser add-ons. The key finding was that all five named extensions, including the one-time-use Secure Exam Proctor, were either inactive or no longer needed for their original purpose. The recommendation was the immediate removal of all unused extensions specifically, <strong>Secure Exam Proctor, Enhancer for YouTube, Copy-n-Paste: Clipboard Upload Simplified, Merlin - Ask AI to Research, Write & Review, and Go Full Page - Full Page Screen Capture.</strong> This proactive step successfully completes the task by eliminating unnecessary software and reducing the "attack surface" of my browser.</p><br>
<br>

Conclusion:<br>

<p align="left">While i removed these extensions because they were unused, it is important to note that any unnecessary extension poses an inherent risk. The core security strategy is the Principle of Least Privilege: minimize the number of installed extensions to only those you absolutely require. This limits the total amount of third-party code running in your browser, which is the key to minimizing security risks. For long-term safety, implement a Quarterly Security Audit to remove any extension you haven't actively used in the last few months. Always scrutinize permissions before installing, as extensions that ask for excessive access (like reading all website data) unnecessarily increase your exposure. For sensitive activities, use a dedicated, extension-free browser profile to completely isolate high-value data from all third-party code.</p><br>
